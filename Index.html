Working Directory

El area de trabajo corresponde a la carpeta donde inicializamos el repositorio de git, rara vez vamos a estar utilizando la linea de comandos y es basicamente donde vamos a crear, editar y eliminar los diferentes archivos de nuestro proyecto.

  • dir || ls muestra el contenido de la carpeta actual
  • mkdir & name crea una carpeta con el nombre especificado
  • cd & path cambia la carpeta actual por la seleccionada
  • git init inicializa un repositorio de git en la carpeta actual
  • rm -rf .gif elimina el repositorio de manera forzada
